# AN0005: Add assertion in test

| Property                    | Value    |
| --------------------------- | -------- |
| Id                          | AN0005   |
| Category                    | Usage    |
| Default Severity            | Warning  |

## Example

### Code with Diagnostic

```csharp
[TestMethod]
public void TestMethodName()
{
        int k = 1 + 2;
}
```

### Code with Fix

```csharp
[TestMethod]
public void TestMethodName()
{
        int k = 1 + 2;
        j.Should().Be(3);
}
```

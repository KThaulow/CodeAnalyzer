# AN0004: Use 24 hour time format

| Property                    | Value    |
| --------------------------- | -------- |
| Id                          | AN0004   |
| Category                    | Formatting    |
| Default Severity            | Warning  |

## Example

### Code with Diagnostic

```csharp
string date = DateTime.Now.ToString("mm/dd/yyyy hh:mm:ss");
```

### Code with Fix

```csharp
string date = DateTime.Now.ToString(""mm/dd/yyyy HH:mm:ss"");
```

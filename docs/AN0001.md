# AN001: Circuit breaker in loop

| Property                    | Value    |
| --------------------------- | -------- |
| Id                          | AN0011  |
| Category                    | Style    |
| Default Severity            | Warning  |

## Example

### Code with Diagnostic

```csharp
bool running = true;
while(running)
{
}
```

### Code with Fix

```csharp
int counter = 0;
bool running = true;
while(running)
{
  counter++;
  if(counter > 10)
    break;
}
```
